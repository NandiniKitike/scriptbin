# ScriptBin

ScriptBin is a lightweight Pastebin-like application built using **Next.js**, **MongoDB**, and **Tailwind CSS**.

## Features
- Create and share text or code snippets
- View pastes using a unique public URL
- REST API for creating and fetching pastes
- MongoDB integration with Mongoose
- Health check endpoint

## Tech Stack
- Next.js (App Router)
- MongoDB + Mongoose
- Tailwind CSS
- TypeScript
